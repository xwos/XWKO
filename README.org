* 简介

玄武内核模块是[[https://gitee.com/starsoul/XuanWuOS][玄武操作系统]]在
Linux内核中的实现。

+ 简称：玄武KO
+ 英文名：XuanWuKO
+ 简称：XWKO

支持的Linux内核版本范围：3.0.x ~ 5.4.x 。

--------

* 源码结构

+ Build
  - xwbs: 编译系统
+ BSP & Project
  - xwcd: 架构、CPU、SOC、外设驱动相关的代码
  - xwbd: 工程配置、工程驱动与工程应用程序
+ OS
  - xwos: 内核
+ Middleware
  - xwmd: 中间件

--------
